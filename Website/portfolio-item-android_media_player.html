Project Purpose and Goal

This is a project made while following a tutorial to familiarize myself more with Android Studio and with the default commands that use the capabilities of the phone. The app lists all the songs found on the internal and external memory of the phone in order to be played.

Problems and Thought Process

Originally the app made after the tutorial had 2 noticeable bugs,one where the app crashes if while a song is paused it is select another, and the other where with a song paused, if it is dragged along the seekbar it will start the song from the beginning. The bugs have been resolved by eliminating a few checks that were impeding the functions of the app without helping.

This project can very well be improved with new functions like a search button or more activities to organize the songs by the user. I plan to upgrade this app as I become more proficient in time, as at this time, the app still has some things to teach me about it.
